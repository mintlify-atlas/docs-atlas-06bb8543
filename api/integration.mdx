## Overview

Integrations define the external services and tools that Flowbaker can connect to. Each integration provides actions and triggers that can be used in workflow nodes.

## Integration Structure

### Go Definition

```go
type Integration struct {
    ID                   IntegrationType
    Name                 string
    Description          string
    CredentialProperties []NodeProperty
    Actions              []IntegrationAction
    Triggers             []IntegrationTrigger
    EmbeddingModels      []IntegrationEmbeddingModel
    IsGroup              bool
    CanTestConnection    bool
    IsCredentialOptional bool
}
```

## Helper Methods

### GetActionByType

Retrieves a specific action from an integration.

```go
func (i Integration) GetActionByType(actionType IntegrationActionType) (IntegrationAction, bool)
```

**Example:**
```go
action, found := integration.GetActionByType("send_message")
if found {
    // Use the action
}
```

## Peekable Data

Many integrations support "peeking" to dynamically load options:

```go
type IntegrationPeekablePaginationType string

const (
    PeekablePaginationType_None   = "none"
    PeekablePaginationType_Cursor = "cursor"
    PeekablePaginationType_Offset = "offset"
)
```

Examples:
- List Slack channels
- List database tables
- List Google Drive folders
- List GitHub repositories

Use the [Peek Data endpoint](/api/executions#peek-data) to retrieve these options.
